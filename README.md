# Design-Pattern---Banking-System
Design Pattern Individual Assignment using Strategy and Factory Patterns

# 1. System Requirements
Java Development Kit (JDK): Version 8 or higher (e.g., JDK 11, JDK 17, or JDK 21).
Integrated Development Environment (IDE): IntelliJ IDEA (Recommended), Eclipse, or NetBeans.
Operating System: Windows, macOS, or Linux.

# 2. Installation Steps
* Step 1: Extract the Project Files
Locate the submitted zipped file named TP065488_SourceCode.zip.
Right-click the file and select Extract All (or use an unzipping tool).
Ensure the extracted folder contains the following sub-directories:
simple_solution/
refined_solution/

* Step 2: Open in IntelliJ IDEA
Launch IntelliJ IDEA.
Click Open or File > Open.
Navigate to the extracted folder (TP065488_SourceCode) and select it
